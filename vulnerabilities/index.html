Common MCP Vulnerabilities

  • SQL Injection (SQLi): Raw SQL execution without proper sanitization
  • Remote Code Execution (RCE): Unrestricted command execution capabilities
  • Authentication Bypass: Missing or weak authentication mechanisms
  • Input Validation Failures: Lack of proper input sanitization
  • Privilege Escalation: Insufficient access controls and role separation
  • Data Exposure: Unmasked PII and sensitive information leakage
  • JSON-RPC Exploitation: Protocol-specific attack vectors

Intentionally Vulnerable MCP Server

Research Environment: We maintain an intentionally vulnerable MCP server for security research and testing purposes. 
# Features of the vulnerable server:
- No authentication or access control
- Raw SQL query execution
